# senmu-vault

## 技术栈

- 框架：React@18.x、web3-react
- 构建工具：webpack、swc
- 测试工具：Jest、BackstopJS(UI)
- 其他：tailwindCss、TypeScript、Yarn

## 如何启动？

> 请确保先执行 `yarn` 安装所有依赖

`yarn start`
